Super Simple to Use

1. Install 

npm install @eva/css

2. Configure with YOUR sizes 

@use '@eva/css' with (
  $sizes: (4, 8, 16, 32, 64),    // YOUR sizes!
  $font-sizes: (16, 24, 32),
  $build-class: true
);

3. Use generated utilities 

<div class="w-64 p-16 _bg-brand">
  <h1 class="fs-32 _c-light">Hello EVA!</h1>
</div>
