EVA CSS - Simple SCSS Example

This example demonstrates using EVA CSS with SCSS variables

Configuration Method: SCSS Variables

This project uses inline SCSS configuration with the @use 'eva-css-fluid' with () syntax. All configuration is defined directly in the styles/main.scss file.

@use 'eva-css-fluid' with ( $sizes: 4, 8, 16, 32, 64, 128, $font-sizes: 14, 16, 20, 24, 32, 48, $build-class: true );

Simple No build script Works immediately

Features Demonstrated

Fluid Sizing

All spacing and font sizes use EVA's fluid clamp() functions. Resize your browser to see elements scale smoothly without media queries.

Utility Classes

EVA generates utility classes like .w-64, .p-16, .fs-24 that you can use directly in HTML.

CSS Variables

Use variables like var(--32) or var(--24) in your custom SCSS for consistent, fluid spacing.
