Build Process Comparison

SCSS Variables

Configuration inline in SCSS:

@use 'eva-css-fluid' with ( $sizes: 4, 8, 16, 32 );

Build with:

npx sass styles/main.scss

JSON Config

Clean SCSS import:

@use 'eva-css-fluid';

Build with custom script:

node scripts/build-with-config.js \ styles/main.scss dist/main.css

Setup Instructions

1. Install Dependencies

npm install

2. Validate Configuration (Optional)

npm run validate

3. Build CSS

npm run build

4. Watch Mode (Auto-rebuild)

npm run watch

This watches for changes to styles/*.scss and eva.config.cjs and automatically rebuilds when you save.

5. Customize Configuration

Edit eva.config.cjs to change sizes, colors, or theme:

module.exports = { sizes: [10, 20, 40, 80], // Your custom sizes fontSizes: [16, 24, 36], // Your font sizes theme: { colors: { brand: '#your-color' } } };

Save the file and the build will automatically update if watch mode is running.
